# OmniInsight AI

Working multi-industry customer intelligence project.

## Run
## Run Project Locally

### 1. Create Virtual Environment
```bash
python -m venv .venv
```

### 2. Activate Environment
```bash
.venv\Scripts\activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run Application
```bash
python app.py
```

### 5. Open in Browser
```txt
http://127.0.0.1:5000
```
